feat: initial release of Restaurant Management System

Integrated solution comprising a mobile app, admin dashboard, and backend API.

Backend (.NET 10 API):
- 40+ REST endpoints for orders, menu, users, etc.
- JWT authentication/authorization with role-based access.
- SignalR for real-time order tracking and admin notifications.
- Enhanced security with hardened CORS and JWT secret externalization.
- Optimized performance with critical database indexes.
- Standardized error handling and detailed request/response logging.

Admin Dashboard (Blazor Server):
- Responsive UI with light/dark mode support.
- KPI analytics and real-time order management.
- Dynamic management pages for categories, menu, and offers.
- Integrated API services with SignalR sync.

Mobile App (Flutter):
- 18+ screens with multi-language (EN/AR) and RTL support.
- Complete ordering flow: menu browsing, cart, and checkout.
- Real-time order tracking and address management.
- User profile and authentication management.

Infrastructure:
- Clean Architecture with Repository pattern.
- Entity Framework Core with SQL Server.
- Migration-ready database schema.
